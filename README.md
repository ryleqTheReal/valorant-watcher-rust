# VALORANT Watcher
  This is the official companion software for valstats.site which collects data about the player and their VALORANT-specific behaviour and sends it to the
  valstats server for evaluation.

  The goal of the project is to have deeper insights into VALORANT player behaviour and be able to share more in-detail things about oneself.

  The background process uses up about ~20 MB of RAM and does NOT affect any rate limits of other apps. It is basically totally silent in that sense.

# Installation
  Install the newest version of Rust.

  1. Clone the project
  2. Switch into the project's directory and run `cargo build --release`
  3. Start the app

Don't forget to make the app auto-start for comfort, unless you want to manually launch it every time, you do you!
